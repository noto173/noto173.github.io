notocities

this is a very epic site to host very epic sites. it is written in node.



Passwords are used to protect your site from getting edited by other people.
If you set one, make sure you have it saved somewhere.
Setting this when you save your website will give it the password you set.
Setting this when editing a pre-existing site that does not have a password will set the password for the site.

PLEASE DO NOT USE A PASSWORD YOU USE ON ANY OTHER SITE. THE PASSWORDS ARE STORED
IN PLAINTEXT ON THE SERVER. THEY ARE NOT SECURE.

Adding a password to your site does NOT make it private. It just makes it so people can't edit it.
